# README

專案目標用戶:
新手爸媽、非嬰幼兒父母的照顧者與暫時陪伴者，以及幼兒本人。

專案特色:
本專案提供快速且符合目標嬰幼兒年齡的單一遊戲建議，首頁設置一鍵產生推薦遊戲按鈕，依照用戶設定之陪伴對象年齡推薦遊戲。
用戶亦可使用孩子年紀、合適情境、主題標籤等類別進行瀏覽，或直接使用關鍵字搜尋。
用戶可將有興趣的遊戲單存入最愛清單，方便直接快速取用，亦可自製遊戲單，分享給其他人。
為了讓尚未習字的幼兒操作本專案，導入簡易的語音啟動一鍵推薦遊戲，以及語音朗讀遊戲單內容，
配合遊戲單中的影音圖片，讓幼兒亦能理解並挑選遊戲。

User Story:
--已完成--
1.用戶/訪客可以點擊"Play!”按鈕，得到隨機遊戲單建議。
2.用戶/訪客可以依照類別(或標籤)、被收藏數、瀏覽特定遊戲。
3.用戶/訪客可瀏覽單個別遊戲，遊戲顯示包含：
	遊戲標題
	遊戲發表者(可連到個人頁)
	遊戲圖片
	適合年齡
	適玩情境
	遊戲道具
	遊戲影片
	遊戲方法
	其他使用者對此遊戲的回應
	用戶可以回應此遊戲


用戶可以創建帳號、登入、登出
	除了信箱和密碼，用戶在註冊時還能設定"陪伴對象"的生日(自動換算成年齡)
	用戶能選擇性編輯自己的名稱、介紹和大頭照，以及"陪伴對象"的生日
	用戶能將遊戲單加入/移出收藏的遊戲清單
	適合用戶設定之陪玩對象年齡的遊戲會"適齡"圓標
	用戶可以發表/編輯/刪除自己創建的遊戲單


用戶可以用語音啟動遊戲推薦功能(限回答1~12歲)
用戶可以利用圖示按鈕，啟動朗讀遊戲單標題與朗讀遊戲單實施步驟
用戶可以利用遊戲標題、遊戲道具與遊戲方法進行關鍵字搜尋

--進行中--
用戶可以對其他用戶留言(悄悄話)
用戶可以依照遊戲標籤進行搜尋
用戶可以依照"陪伴的孩子年齡"查看其他用戶訊息
用戶可以關注其他用戶，可以查看自己關注的對象與關注自己的人
